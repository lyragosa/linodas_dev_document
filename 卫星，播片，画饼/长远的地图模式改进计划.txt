仅为E3展示，实际效果请参考其他在E3上展示的厂商

新的卫星中，现在的只在N个城市之间切换的地图， 改为“大地图模式”。
因为无美工，大地图可能是RPGMAKER制作/白板/地铁线路图风格/手绘风格

# 城市
# 同现在一样，但城市的具体地区之间切换都需要移动，城市内移动不消耗AP
# 除此之外，城郊的点需要消耗AP才能到。
 + 下列是一些例子
 + 卡拉林 -> 卡拉林郊外草药采集地  1AP
   当玩家正在移动的时候不能进行动作，但可以进行其他操作
 + 当移动到野外后依然可以访问商店，但不能进行购买。
 + 卡拉林 -> 卡拉林峡谷  5AP
  - 需要这么多AP才能抵达副本门口（从地图上移动到这个位置）
  - 每一关需要 20 AP
 + 大地图地区
  - 例如将卡拉林分为下述城市区域
   - 喷水池广场 
   - 闹市区
   - 旧城区
   - 学院区
   - 空港
  - 玩家切换到不同区域需要花费AP
 + 因为AP消耗在这种模式下会大幅增加，所以会大幅提高玩家获得的AP量，保持和以前持平。
 + 地图上会给予大量随机刷新的位置，踩中这些位置会给AP
 + 一些偏僻角落，和需要很多AP才能前往的格子，可能会安置宝箱，或者特殊的NPC，或者定时开催的副本。
# 野外
 + 除了城市，现在玩家还可以移动到各种野外地区
 + 例如，从艾尔玛出来，可以移动到城市野外附近的下列区域
  - 艾尔玛下城区 -> 对应有数个NPC可交谈+若干个商人
  - 石碑遗址 -> 就地可以探索出一个场景战役
  - 灰暮洞穴 -> 一个固定的场景战役 + 若干NPC
  - 泰伦河 -> 一些商人
  - 古代龙巢穴 -> LV15团本
  - 旅行者法罗的小屋 -> 不一定出现的可交谈NPC
 + 在真·野外，也就是需要消耗大量AP才能移动到的位置
  （请注意，城市和城市间是鸟点移动，因此只消耗移动力而不消耗AP）
  - 北方山脉大地图
   - 落雪洞窟 -> 过路
   - 荒废的尖塔 -> 特定NPC
   - 探险家协会营地 -> 一系列商人
   - 被深埋的祭坛 -> 随机出现签契约的NPC
  - 艾斯提亚和新月堡之间的大地图
   - 沙漠驿站 -> 随机NPC 过路 一些事件
   - 绿洲 -> 可以吃到回复AP的道具 立刻回复
   - 深陷的沙丘 -> 可能会被陷入沙丘里然后被踢回炉石点，也可能发现稀有道具
   - 沙漠小镇 -> 商人和稀有商人
